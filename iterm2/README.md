## Keyboard Shortcuts

### Switch tabs

| Shortcut | Action |
|----------|--------|
| `⌘ →` | Next Tab |
| `⌘ ←` | Previous Tab |
| `⌘ ⇧ →` | Move Tab Right |
| `⌘ ⇧ ←` | Move Tab Left |

### Split panes

| Shortcut | Action |
|----------|--------|
| `⌘ D` | Split pane vertically |
| `⌘ ⇧ D` | Split pane horizontally |
| `⌘ [` / `⌘ ]` | Switch between panes |

### Jump forwards/backwards words

`iTerm2` > `Settings` > `Profiles` > `Keys` > `Key Mappings` > `+`:

| Shortcut | Action | Esc+ |
|----------|--------|------|
| `⌥ ←` | Send Escape Sequence | `b` |
| `⌥ →` | Send Escape Sequence | `f` |
